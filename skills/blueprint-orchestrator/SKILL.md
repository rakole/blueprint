# Blueprint Orchestrator

Blueprint is a structured delivery workflow for Gemini CLI.

Use this skill when the task needs coordination across phases rather than a one-off answer.

## Workflow

1. Clarify the outcome and constraints.
2. Break work into small execution slices.
3. Keep the active slice narrow and verifiable.
4. Review the result before declaring completion.

## Operating rules

- Prefer incremental delivery over large speculative changes.
- State assumptions when details are missing.
- Keep plans concrete enough to execute immediately.
- Preserve repository conventions and existing code patterns.
